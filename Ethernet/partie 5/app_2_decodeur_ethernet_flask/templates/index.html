Décodage d'une trame Ethernet

Choisissez le type d'oscilloscope et chargez le fichier CSV exporté. L'application va : mise en forme du signal → décodage Manchester → décodage Ethernet.

Le type est utilisé pour choisir le bon format de CSV (colonne, ligne de début, etc.).
Fichier exporté directement depuis l'oscilloscope (sans modification).
